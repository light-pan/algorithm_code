```
给定M（0<M<=30）个亨符（a-z），从中取出任意字符（每个字符只能用一次）拼接成长度为N（O<N<=5）的字符串，要求相同的字符不能相邻，计算出给定的字符列表能拼接出多少种满足条件的字符串，输入非法或者无法拼接出满足条件的字符串则返回0。
```

### 输入描述：
```
给定的字符列表和结果字符串长度，中间使用空格（" "）拼接
```

### 输出描述：
```
满足条件的字符串个数
```

### 示例1
#### 输入
```
abc 1
```

#### 输出
```
3
```
#### 说明
```
给定的字符为a.b，c，结果字符串长度为1，可以拼接成a，b，c，共3
```
